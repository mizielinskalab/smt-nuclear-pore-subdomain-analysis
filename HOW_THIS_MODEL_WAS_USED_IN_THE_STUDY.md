# How this model was used in the study

This model was used as a **supporting image-analysis component** in the study.

## Role in the workflow

1. Transmitted-light bright-field images were used as model input.
2. The study-specific Lamin B1 model predicted Lamin B1-like fluorescence patterns from those images.
3. The predicted Lamin B1 signal was then used to assist the definition of the nuclear-envelope region for downstream analysis.

## What this repository is meant to document

This repository is intended to document:

- the exact Lamin B1 model used,
- the type of input images provided to the model,
- the output files generated by inference,
- the relationship between input, prediction, and measured Lamin B1 images.

## What this repository is not meant to claim

This repository does not claim that the deep-learning model is the main technical contribution of the paper. The biological conclusions of the paper do not depend on presenting this model as a new general framework.
